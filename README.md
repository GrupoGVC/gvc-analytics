# GVC Analytics

Sistema de Dashboard Analítico desenvolvido para o Grupo GVC, com foco em visualização estratégica de indicadores, monitoramento operacional e apoio à tomada de decisão.

## Sobre o Projeto

O GVC Analytics é uma plataforma web criada para centralizar informações estratégicas e operacionais das empresas do Grupo GVC, permitindo uma visão integrada dos dados através de dashboards intuitivos e interativos.

O projeto foi desenvolvido com foco em:

- Indicadores operacionais
- Gestão de resíduos
- Monitoramento de desempenho
- Acompanhamento das unidades
- Análises gerenciais
- Visualização de métricas estratégicas

---

## Sobre o Grupo GVC

O Grupo GVC atua na gestão, valorização e tratamento de resíduos, promovendo soluções sustentáveis e inovadoras para o setor ambiental.

O ecossistema do grupo contempla operações voltadas para:

- Valorização de resíduos
- Compostagem
- Beneficiamento de agregados reciclados
- Produção de biogás
- Tratamento de chorume
- Soluções ambientais sustentáveis

Site institucional:  
https://grupogvc.eco.br/
